Les trois commandes de bases de Git

Git add monfichier

Ajoute à l'index (stage) les nouveaux fichiers ou les nouvelles modifications du fichier modifié

Git commit

Ajoute une nouvelle version

Git push origin main

Ajoute sur GitHub les nouvelles versions

Créer des branch pour faire les modifications à un projet existant

git branch

montre la liste des branch, celle portant une étoile est celle qu'on modifie

git branch nouvelleOption

créé une nouvelle branch pour développer de nouvelles fonctionnalités

git checkout nouvelleOption

permet d'aller dans le "répertoire" pour y effectuer les développements

git commit -m " développement de nouvelleOption nouvelle branch"

une fois finalisée : On commit pour créer la nouvelle version

git checkout main

retourne au dossier main afin de fusionner

git merge nouvelleOption

fusion dans le dossier main
